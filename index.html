Veille Technologique


Qu'est ce que la veille technologique ?


La veille technologique consiste à surveiller les évolutions techniques et les innovations dans un secteur d'activité donné. Cela comprend notamment la surveillance, la collecte, le partage et la diffusion d'information permettant d'anticiper ou de s'informer sur des changements en matière de recherche, développement, matériaux, processus, etc...

REACT

React (aussi appelé React.js ou ReactJS) est une bibliothèque JavaScript libre développée par Facebook depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d’application web monopage via la création de composants dépendant d’un état et générant une page (ou portion) HTML à chaque changement d’état. La syntaxe de React est le JSX qui est un mixe de HTML et Javascript,un point majeure qui permet de faire un composans contenant du javascript et du html en un unique fichier.

React est déclarative efficace et flexible pour construire des interfaces utilisateurs (UI).Elle permet de composer des UI complexes à partir de petit morceaux de code isolés appelés « composants », elle a plusieurs type de composants. Ne gérant que l’interface de l’application elle est considérée comme la vue dans le modèle MVC. Elle peut ainsi être utilisée avec une autre bibliothèque ou un Framework MVC comme AngularJS. La bibliothèque est utilisé par Netflix, Yahoo, Sony, Atlassian ainsi que par les équipes de Facebook, Instagram ou encore WattsApp. Même Wordpress.com annonce Gutenberg, une interface développée en JavaScript avec Node.js et React. A cela s’ajoute Instagram,Paypal, Netflix, DropBox, Walmart, Airbnb, Deezer, Ebay, Periscope, Pinterest, Twitter Mobile. React a été conçu dès le départ pour une adoption progressive, et on peut l’utiliser aussi peu ou autant de React qu’on le souhaite. Avec ce dernier on peut ajouter de l’interactivité à une simple page HTML ou démarrer une application complexe alimentée par React, on peut créer une nouvelle application, on peut l’ajouter à un site web, on peut aussi jouer avec React en utilisant un terrain de jeu de code en ligne. La technologie React.js se base sur des composants. Ces composants peuvent ensuite être réutilisés facilement et rapidement pour d’autres projets ou à l’intérieur même du projet. Il faut savoir et comprendre, qu’aujourd’hui, l’utilisation des composants est le futur du développement web. La grande force de la librairie ReactJS est qu’elle s’intègre parfaitement à d’autres technologies, CMS et frameworks. de nombreuses mises à jour sortent régulièrement.Avec des outils tels que React Native,on peu constater une évolution perpétuel qui à un impact directe sur les entreprises. Avec le React les streamers dépoussièrent les émissions télé sur Twitch-konbini. On peut concevoir un clip d'application efficace avec React Native. Les App Clips sont une fonctionnalité de la plateforme iOS et faire des configurations du routeur React en executant les commandes.
