Veille Technologique

REACT

React (aussi appelé React.js ou ReactJS) est une bibliothèque JavaScript libre développée par Facebook depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d’application web monopage via la création de composants dépendant d’un état et générant une page (ou portion) HTML à chaque changement d’état.

React est déclarative efficace et flexible pour construire des interfaces utilisateurs (UI).Elle permet de composer des UI complexes à partir de petit morceaux de code isolés appelés « composants », elle a plusieurs type de composants. Ne gérant que l’interface de l’application elle est considérée comme la vue dans le modèle MVC. Elle peut ainsi être utilisée avec une autre bibliothèque ou un Framework MVC comme AngularJS. React a été conçu dès le départ pour une adoption progressive, et on peut l’utiliser aussi peu ou autant de React qu’on le souhaite. Avec ce dernier on peut ajouter de l’interactivité à une simple page HTML ou démarrer une application complexe alimentée par React, on peut créer une nouvelle application, on peut l’ajouter à un site web, on peut aussi jouer avec React en utilisant un terrain de jeu de code en ligne.

Dernières fonctionnalités

• Suspense pour les données asynchrones
• Mises à jour partielles
• API de rendu en continu
• Support de l'auto-suspension
• Nouveaux hooks et utilitaires

Impact:
• Amélioration de la performance
• Simplification du développement
• Meilleure expérience utilisateur
• Code plus propre et plus maintenable
En résumé, React JS 18 permet de créer des applications web plus performantes, plus réactives et plus faciles à utiliser.

les versions de REACT

React JS évolue régulièrement, apportant de nouvelles fonctionnalités et améliorations. Voici un aperçu des versions importantes et ce qu'elles proposent :
Version actuelle : React 18
Sortie en mars 2023.
Principales nouveautés : introduction de l'API racine en remplacement de ReactDOM.render().

Version précédente : React 17
Sortie en octobre 2020.
Principales nouveautés : introduction de la fonctionnalité "suspense" permettant de charger des composants dynamiquement de manière asynchrone, et compatibilité accrue avec les navigateurs web modernes.
Versions antérieures :
React 16: Introduit des composants fonctionnels, le context API mis à jour et une meilleure infrastructure interne.
React 15: Introduit les classes de composants, le cycle de vie des composants et le référencement sécurisé.

REACT 18

API racine : Remplace ReactDOM.render() pour une gestion plus flexible des rendus et une meilleure intégration avec les frameworks web. Meilleure gestion des rendus : Réduit les re-rendus inutiles pour améliorer les performances et la fluidité de l'interface utilisateur. Transitions et animations : Facilite la création d'interfaces utilisateur animées et interactives.

Nouveaux hooks : useId pour générer des identifiants uniques et useDeferredValue pour optimiser les performances en différant le rendu de certaines valeurs. Améliorations du serveur : Améliore le streaming côté serveur et la prise en charge du rendu côté serveur pour les applications web. Autres fonctionnalités notables : Suspense : Permet de charger des composants dynamiquement de manière asynchrone. Context API mis à jour : Offre une meilleure gestion des contextes et facilite la communication entre les composants. Composants fonctionnels : Offrent une syntaxe plus concise et plus facile à apprendre. Classes de composants : Offrent plus de flexibilité et de puissance pour les cas d'utilisation complexes.
